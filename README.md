## Get Started
### 1. Clone this repository
```bash 
git clone https://github.com/UGoingNoWhereBoy/FaceMatcher-Frontend/
```
### 2. cd into the repository
```bash
cd FacecMatcher-Frontend
```
### 3. Install npm packages
```bash
npm install
```
### 4. Run the app
```bash
npm run dev
```
